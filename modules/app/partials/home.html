Description

famous-flex-angular is an angular module dependent on famous-angular. It uses the $famous service to register the famous-flex components, and $famousProvider to relate the famous render tree to angular scope.

Examples

  • date-picker. Uses the ffa-date-picker directive.
    • currently there is some issue with rotation.
  • flex-scroll-view. Uses the ffa-flex-scroll-view directive. Possible to alter layout function.
    • may need to resize the browser window initially to view the functionality
    • currently layout function selection is wired in, but layout function options only valid for collection layout.
    • can add additional surfaces with the Add Square button
  • grid-layout. Uses the ffa-grid-layout directive. The layout function used is flex-grid-layout.
    • this uses a fixed layout function
  • layout-controller
  • nav-bar. Uses the ffa-nav-bar directive. Transcludes html fragment and wraps buttons in surfaces.
    • some issue with title layout
  • tab-bar. Uses the ffa-tab-bar directive.
    • seems to work fine.
